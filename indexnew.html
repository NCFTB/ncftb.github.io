What is BOINC?

BOINC, a main aspect of NCFTB, is a cloud computing program developed by the University of California, Berkeley. The basic concept is that scientific organizations in need of computational power can use BOINC. And what boinc does is utilize unused performance on any computer running it, combining the power of all these computers to get real work done! BOINC

Systems

All systems run Lubuntu a lightweight Linux Operating System to decrease base system resource usage and allocate the most possible to BOINC. The systems are also all password protected and are kept locked to prevent the unlikely situation in which theft would occur.
